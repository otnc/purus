## 前提条件

:::caution[Node.js v20 以降が必要]
Purus **v0.10.x 以降** は **Node.js v20 以降** が必要です。それ以前のバージョンはサポートされていません。

Purus **v0.9.x 以前** を使用する場合は、Node.js v18 以降が必要です。
:::

以下のいずれかから Node.js をインストールしてください:

- [Node.js v20](https://nodejs.org/ja/download/archive/v20) — v20 各リリース
- [Node.js LTS](https://nodejs.org/en/download) — 最新 LTS リリース
- [nvm](https://github.com/nvm-sh/nvm) *(推奨)* — Node Version Manager（macOS / Linux）
- [nvm-windows](https://github.com/coreybutler/nvm-windows) *(推奨)* — Node Version Manager（Windows）

:::tip[nvm を使う場合]
```sh
# Node.js v20 をインストールして使用
nvm install 20
nvm use 20

# または最新 LTS をインストールして使用
nvm install --lts
nvm use --lts
```
:::

## npmでインストール

最新バージョン: <NpmVersion package="purus" /> — [npmで見る](https://www.npmjs.com/package/purus)

```sh
npm install -g purus
```

## 確認

```sh
purus version
```

## 最初のプログラム

`main.purus` を作成:

```purus
const message be //;Hello, World;//
console.log[message]
```

コンパイルして実行:

```sh
purus build main.purus
node main.js
```

またはファイルを生成せずに直接実行:

```sh
purus run main.purus
```

## プロジェクトを作成

`purus new` で設定、スクリプト、依存関係を含むプロジェクトをセットアップ:

```sh
purus new my-project
cd my-project
npm run build
```
