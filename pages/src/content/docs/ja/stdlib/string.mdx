`string` モジュールは文字列操作のためのユーティリティ関数を提供します。

```purus
use p-string as s
```

## クエリ

### `len[str]`
文字列の長さを返します。

```purus
use p-string as s
s.len[///hello///]  -- 5
```

### `contains[str; sub]`
`str` が `sub` を含んでいれば `true` を返します。

```purus
use p-string as s
s.contains[///hello world///; ///world///]  -- true
```

### `startswith[str; prefix]`
`str` が `prefix` で始まれば `true` を返します。

### `endswith[str; suffix]`
`str` が `suffix` で終われば `true` を返します。

```purus
use p-string as s
s.startswith[///hello///; ///he///]  -- true
s.endswith[///hello///; ///lo///]    -- true
```

### `indexof[str; sub]`
`str` 内で最初に `sub` が出現するインデックスを返します。見つからない場合は -1。

```purus
use p-string as s
s.indexof[///hello///; ///ll///]  -- 2
s.indexof[///hello///; ///xyz///] -- -1
```

### `count[str; sub]`
`str` 内の `sub` の重複しない出現回数を返します。

```purus
use p-string as s
s.count[///banana///; ///an///]  -- 2
```

## 変換

### `upper[str]`
文字列を大文字に変換して返します。

### `lower[str]`
文字列を小文字に変換して返します。

```purus
use p-string as s
s.upper[///hello///]  -- "HELLO"
s.lower[///HELLO///]  -- "hello"
```

### `capitalize[str]`
文字列の最初の文字を大文字にして返します。

### `title[str]`
各単語の最初の文字を大文字にして返します。

```purus
use p-string as s
s.capitalize[///hello world///]  -- "Hello world"
s.title[///hello world///]       -- "Hello World"
```

### `trim[str]`
両端の空白を除去します。

### `trimstart[str]`
先頭の空白を除去します。

### `trimend[str]`
末尾の空白を除去します。

```purus
use p-string as s
s.trim[///  hello  ///]       -- "hello"
s.trimstart[///  hello  ///]  -- "hello  "
s.trimend[///  hello  ///]    -- "  hello"
```

### `reverse[str]`
文字列を反転して返します。

```purus
use p-string as s
s.reverse[///hello///]  -- "olleh"
```

### `repeat[str; n]`
文字列を `n` 回繰り返して返します。

```purus
use p-string as s
s.repeat[///ab///; 3]  -- "ababab"
```

### `replace[str; old; new]`
`old` の**すべて**の出現を `new` に置換します。

### `replacefirst[str; old; new]`
`old` の**最初**の出現を `new` に置換します。

```purus
use p-string as s
s.replace[///aabaa///; ///a///; ///x///]       -- "xxbxx"
s.replacefirst[///aabaa///; ///a///; ///x///]   -- "xabaa"
```

### `padstart[str; len; fill]`
文字列の先頭を `len` に達するまでパディングします。`fill` 文字のデフォルトはスペースです。

### `padend[str; len; fill]`
文字列の末尾を `len` に達するまでパディングします。`fill` 文字のデフォルトはスペースです。

```purus
use p-string as s
s.padstart[///42///; 5; ///0///]  -- "00042"
s.padend[///hi///; 5; ///.///]    -- "hi..."
```

## 分割 / 結合

### `split[str; sep]`
文字列を `sep` で分割し、配列を返します。

### `lines[str]`
文字列を改行で分割します。

### `words[str]`
文字列を空白で分割し、空のセグメントを無視します。

### `join[arr; sep]`
文字列の配列を `sep` で結合します。

### `chars[str]`
個々の文字の配列を返します。

```purus
use p-string as s
s.split[///a,b,c///; ///,///]   -- ["a", "b", "c"]
s.words[///hello  world///]     -- ["hello", "world"]
s.join[[///a///; ///b///]; ///-///]  -- "a-b"
s.chars[///hi///]               -- ["h", "i"]
```

## スライス

### `slice[str; start; end]`
`start` から `end`（排他的）までの部分文字列を返します。

### `charat[str; i]`
インデックス `i` の文字を返します。

### `codeat[str; i]`
インデックス `i` のUnicodeコードポイントを返します。

### `fromcode[code]`
指定されたUnicodeコードポイントの文字を返します。

```purus
use p-string as s
s.slice[///hello///; 1; 3]   -- "el"
s.charat[///hello///; 0]     -- "h"
s.codeat[///A///; 0]         -- 65
s.fromcode[65]               -- "A"
```
