## インストール

```sh
npm install -D prettier @puruslang/prettier-plugin-purus
```

## 使い方

### CLI

```sh
npx prettier --plugin @puruslang/prettier-plugin-purus --write "**/*.{purus,cpurus,mpurus}"
```

### 設定ファイル

`.prettierrc` に追加:

```json
{
  "plugins": ["@puruslang/prettier-plugin-purus"],
  "tabWidth": 2
}
```

## フォーマット内容

- **インデント**: 一貫したインデント（スペースまたはタブ）に正規化
- **スペース**: トークン間の空白を正規化
- **ブラケット**: 関数呼び出しの `[]` 内のスペースなし
- **区切り文字**: `,` と `;` の後にスペースを確保
- **末尾空白**: 末尾のスペースを削除
- **末尾改行**: ファイル末尾に改行を確保
- **文字列**: `//;...;//` および `//;...;//` 文字列の内容はそのまま保持（整形されません）

## 例

フォーマット前:

```purus
const x   be   42
fn greet   name
    console.log[  name  ]
```

フォーマット後:

```purus
const x be 42
fn greet name
  console.log[name]
```
