  <Card title="Shiftキー不要" icon="star">
    `{}()=+*` のような記号を `[]`、`be`、`add`、`mul` のような単語に置き換えます。
  </Card>
  <Card title="JavaScriptにコンパイル" icon="rocket">
    Purusで書いて、JavaScriptに変換。JavaScriptが動く場所ならどこでも使えます。
  </Card>
  <Card title="インデントベース" icon="pencil">
    波括弧なし。CoffeeScriptのようにインデントでブロックを定義します。
  </Card>
  <Card title="充実のツール" icon="setting">
    Linter、Prettierプラグイン、VS Code拡張機能を同梱。
