<CardGrid>
  <Card title="No Shift Key" icon="star">
    Purus replaces symbols like `{}()=+*` with words like `[]`, `be`, `add`, `mul`.
  </Card>
  <Card title="Compiles to JavaScript" icon="rocket">
    Write Purus, get JavaScript. Use it anywhere JavaScript runs.
  </Card>
  <Card title="Indent-based" icon="pencil">
    No curly braces. Blocks are defined by indentation, like CoffeeScript.
  </Card>
  <Card title="Full Tooling" icon="setting">
    Linter, Prettier plugin, and VS Code extension included.

    [VS Code Extension](https://marketplace.visualstudio.com/items?itemName=otoneko1102.purus) · [Linter](https://www.npmjs.com/package/@puruslang/linter) · [Prettier Plugin](https://www.npmjs.com/package/@puruslang/prettier-plugin-purus)
